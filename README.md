# verilog_to_txt_convertor_V1_3
This is a small utility that batch converts .v files into .txt files, making it easier to feed them into LLMs that don’t support .v format. You can either run the Python script or simply download the .exe file and double‑click to run it. For more details, please refer to the README file. 是一个批量把.v文件转换为.txt文件的小工具
